# FLASH

This is script for flashcards.

## Installation
Just clone the repo:
```bash
git clone https://github.com/youshitsune/flash
```

## Usage
Run it:
```bash
python3 main.py <path_to_file>
```

## Introduction to file syntax
To write flashcard just do this:
```txt
Question-Answer;
```
**DISCLAIMER: DON'T USE ; ON LAST FLASHCARD.**
